# Задание номер 5 по курсу информационных технологий (.net)
## <b>_Класс «Словарь» (Dictionary)_</b><br><br><b>Свойства</b>
Количество элементов, есть ли элементы (IsEmpty), массив ключей, массив значений, массив пары ключ-значение (создать для этого соответствующую структуру), индексатор, возвращающий элемент по ключу.<br>
## <b>Методы</b>
Добавить пару ключ и значение, удалить элемент по ключу, проверить, есть ли элемент с заданным ключом в словаре (возвращает true, если элемент с таким ключом есть, иначе - false ), очистить словарь.<br>
Способ хранения данных в словаре не важен, можно использовать массив или список (List), ключ должен быть уникальным, значения могут повторяться.<br><br>
